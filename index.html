RockLang

Язык программирования для тех, кто играет по своим правилам. Обратная совместимость с C++, но с бунтарским духом.

Встроенный IDE

Пишите код без переключения между приложениями. Наш встроенный редактор - это всё, что вам нужно.

C++ Совместимость

Используйте все возможности C++ с более простым и выразительным синтаксисом RockLang.

Компиляция с MinGW

RockLang C++ Эквивалент Описание
use #include Подключение библиотек
var [тип] Объявление переменной
cast static_cast Приведение типов
input std::cin >> Ввод данных
output std::cout << Вывод данных
func auto Объявление функции
class class Объявление класса
? if Условный оператор
! else Альтернативное условие
HOI auto Автоматическое определение типа
Start // Start of main Начало основной программы
End // End of main Конец основной программы
. :: Оператор области видимости
global namespace Пространство имён
